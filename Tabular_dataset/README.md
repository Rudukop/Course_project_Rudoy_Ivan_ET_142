Выбранный кейс: №58 Прогнозирование стоимости медицинской страховки (ОГБУ «ЧРЦНИТ»)

Выбранный датасет: Medical Insurance Cost Prediction

Общая информация:
Название: Medical Insurance Cost Prediction
Источник: Kaggle - Medical Insurance Cost Prediction (https://www.kaggle.com/datasets/harishkumardatalab/medical-insurance-price-prediction)
Авторы: Miri Choi и др. (данные из американской страховой компании)
Дата подготовки: 2018-2022 (разные версии)
Лицензия: CC0: Public Domain (общественное достояние, можно использовать свободно).
Мощность датасета: 1338 строк.

Краткое описание содержимого:
Тип задачи: Регрессия (предсказание непрерывного значения).
Входные данные: Характеристики клиента.
Выходные данные: Стоимость страховки (charges).

Перечень признаков:
age (числовой): возраст основного бенефициара. Диапазон: 18-64.
sex (категориальный): пол страхового подрядчика. Значения: male (мужской), female (женский).
bmi (числовой): индекс массы тела. Диапазон: 15.96 - 53.13. Характеризует вес тела (ИМТ = вес/рост²).
children (числовой): количество детей, охваченных страховкой / количество иждивенцев. Диапазон: 0-5.
smoker (категориальный): является ли человек курильщиком. Значения: yes (да), no (нет).
region (категориальный): регион проживания в США. Значения: northeast, northwest, southeast, southwest.
charges (целевой признак) (числовой): индивидуальные медицинские расходы, выставленные страховкой.
