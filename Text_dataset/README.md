Выбранный кейс: №57 Классификация спама по тексту электронного письма (ОГБУ «ЧРЦНИТ»)

Выбранные датасеты: Spam or Ham (SMS/Email) Dataset

Общая информация:
Название: SMS Spam Collection Dataset
Источник: UCI Machine Learning Repository (https://archive.ics.uci.edu/dataset/228/sms+spam+collection) и Kaggle - SMS Spam Collection (https://www.kaggle.com/datasets/uciml/sms-spam-collection-dataset)
Авторы: Tiago A. Almeida, José María Gómez Hidalgo.
Дата подготовки: 2012.
Лицензия: Свободное использование для академических целей.

Описание содержимого:
Тип задачи: Классификация текста.
Входные данные: Текст SMS-сообщения.
Выходные данные: Метка класса (spam или ham).
Количество: 5574 SMS-сообщения на английском языке (4827 ham, 747 spam).

Перечень признаков:
v1 (категориальный): Метка. Значения: ham, spam. (целевой признак)
v2 (текст): Текст сообщения.
Пропуски: Отсутствуют.

Чувствительность: Данные не являются конфиденциальными, так как сообщения были специально собраны для публичного использования.
