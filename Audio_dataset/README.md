Выбраннй кейс №67: Транскрибация и протоколирование контрольного мероприятия

Общая информация
Название датасета: SOVA — Russian Speech Dataset
Источник: Hugging Face — sovaai/sova-dataset(https://github.com/sovaai/sova-dataset)
Авторы: SOVA.ai
Дата подготовки: 2021-2023
Лицензия: CC BY-NC 4.0 (некоммерческое использование с указанием авторства)
Тип задачи: Автоматическое распознавание речи (ASR) — транскрибация аудио в текст
Объем датасета: Более 200 часов русской речи

Описание содержимого
Тип задачи: Speech-to-Text (Seq2Seq) — преобразование аудиосигнала в текстовую расшифровку.
Входные данные: Аудиофайлы (WAV, 16 kHz, моно)
Выходные данные: Текстовая расшифровка
Количество аудиофайлов: ~50,000 записей
Распределение по длительности:
1-5 секунд: 30%
5-10 секунд: 45%
10-20 секунд: 20%
20 секунд: 5%

Демографическое распределение:
Мужской голос: ~45%
Женский голос: ~55%
Разные возрастные группы
