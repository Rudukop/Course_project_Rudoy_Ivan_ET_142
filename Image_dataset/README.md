Выбранный кейс: №48 Распознавание средств индивидуальной защиты (СИЗ) (ЧКПЗ)

Выбранный датасет: Hard Hat Detection Dataset

Общая информация:
Название: Hard Hat Detection Dataset
Источник: Kaggle - Hard Hat Detection Dataset(https://www.kaggle.com/datasets/andrewmvd/hard-hat-detection) и  Roboflow Universe - Hard Hat Workers (https://universe.roboflow.com/joseph-nelson/hard-hat-workers)
Авторы: Andrew Maranhão (оригинальный датасет), Roboflow (обработка и публикация)
Дата подготовки: 2020-2022 (разные версии)
Лицензия: CC BY 4.0 (Creative Commons Attribution 4.0 International) — разрешает использование с указанием авторства.
Мощность датасета: ~5,000 изображений с аннотациями.

Описание содержимого:
Тип задачи: Детекция объектов (object detection).
Входные данные: Изображения со строительных площадок, производственных цехов (формат JPEG, PNG).
Выходные данные: Координаты ограничивающих рамок (bounding boxes) вокруг объектов и метки классов.
Количество изображений: Всего ~5000, обычно разделенных на обучающую (train), валидационную (val) и тестовую (test) выборки (например, 70/15/15).
Количество объектов: Более 15,000 размеченных объектов.
Перечень классов:
helmet — строительная каска надета на голову.
head — голова человека без каски.
person — человек (используется не во всех версиях датасета, иногда только helmet и head).
