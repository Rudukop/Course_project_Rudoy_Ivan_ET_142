Выбранный кейс: №62 Диагностика подшипников с помощью гибридной CNN-MLP модели (ПАО «Северсталь»)

Выбранный датасет: CWRU Bearing Data Center

Общая информация:
Название: Case Western Reserve University (CWRU) Bearing Data Center
Источник: Case Western Reserve University - Bearing Data Center (https://engineering.case.edu/bearingdatacenter/12k-drive-end-bearing-fault-data)
Авторы: Кеннет Лопаро (Kenneth Loparo), профессор CWRU.
Дата подготовки: Начало 2000-х годов, но до сих пор активно используется.
Лицензия: Свободное использование для академических и исследовательских целей.

Описание содержимого:
Тип задачи: Классификация временных рядов (или классификация сигналов).
Входные данные: Временные ряды вибрационных сигналов, собранных с ускорениемеров.
Выходные данные: Состояние подшипника (норма, дефект внешнего кольца, дефект внутреннего кольца, дефект тела качения).

Ключевые особенности:
Данные собраны на экспериментальном стенде с электродвигателем.
Подшипники тестировались в нормальном состоянии и с искусственно созданными дефектами разного диаметра (0.007, 0.014, 0.021 дюйма).
Объем: Огромное количество записей на разных частотах дискретизации (12 кГц и 48 кГц) и под разной нагрузкой.
Пропуски: Отсутствуют.
